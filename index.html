Procedural Low-Poly Tree Generator

Built with Godot 3 · Python Scripting · Real-Time Customisation

Project Overview

The Procedural Low-Poly Tree Generator generates fully customisable low-poly trees in real time. Users can adjust branches, leaves and fruit parameters and instantly visualise the results.

Designed for game developers, technical artists and 3D creators, the tool provides both flexibility and performance for various creative workflows.

Key Features

  • Real-time procedural mesh generation
  • Adjustable branch, leaf and fruit parameters
  • Low-poly stylised aesthetic
  • Custom preset saving to local PC
  • .OBJ export support

Applications

  • Game development
  • Animation and cinematics
  • Architectural visualisation
  • 3D printing workflows

About the Creator

My name is Alan Harszczyk. I created this project as the culmination of my four-year study, serving as my final year project.

My goal was to build a tool that can assist designers — whether in art, games or other creative workflows — while also challenging myself technically. This project allowed me to explore procedural generation, real-time 3D systems and Python scripting in Godot 3.
